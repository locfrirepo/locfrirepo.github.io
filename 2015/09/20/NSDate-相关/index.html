1. 创建或初始化可用以下方法

用于创建NSDate实例的类方法有:
var date = NSDate()
返回当前时间
var date = NSDate(timeIntervalSinceNow: 1000)
返回以当前时间为基准，然后过了1000秒的时间
var date = NSDate(timeIntervalSinceReferenceDate: 1000)
返回以2001/01/01 GMT为基准，然后过了1000秒的时间
var date = NSDate(timeIntervalSince1970: 1000)
返回以1970/01/01 GMT为基准，然后过了1000秒的时间
var date:NSDate = NSDate.distantFuture() as! NSDate
返回很多年以后的未来的某一天。（比如你需要一个比现在(Now)晚(大)很长时间的时间值，则可以调用该方法。测试返回了4000/12/31 16:00:00）
var date:NSDate = NSDate.distantPast() as! NSDate
返回很多年以前的某一天。（比如你需要一个比现在(Now)早(小)大很长时间的时间值，则可以调用该方法。测试返回了公元前0001/12/31 17:00:00）
